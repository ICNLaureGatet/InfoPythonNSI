Introduction à python

Python est un langage de programmation caractérisé par des blocs marqués par des indentations (touche tabulation).
Le respect de cette indentation est primordiale pour assurer la compilation du code.

Les variables

Définition

Une variable est une donnée de votre programme, stockée dans votre ordinateur.

En Python, pour donner une valeur à une variable, il suffit d'écrire nomVariable = valeur.
Le symbole = est souvent représenté en langage courant par une flèche gauche ←.

Quelques règles à respecter :

  1. le nom de la variable ne peut être composé que de lettres, majuscules ou minuscules, de chiffres et du symbole souligné « _ » (appelé underscore en anglais) ;
  2. le nom de la variable ne peut pas commencer par un chiffre ;
  3. le langage Python est sensible à la casse, ce qui signifie que des lettres majuscules et minuscules ne constituent pas la même variable (la variable AGE est différente de aGe, elle-même différente de age).

Les types de données

Une variable peuvent représenter :

  • un nombre (entier - int - ou en virgule flottante - float) sur lequel on peut effectuer des opérations :
    • opérations élémentaires (addition, soustraction, multiplication, division) : +, -, *, /
    • élever à une certaine puissance : a**n pour «a puissance n»
    • division entière sur les entiers : a//b et reste dans la disvision entière : a%b
  • une chaîne de caractères : toujours entre guillemets ou côtes
    Par exemple ch="Mon premier texte" ou ch2='Mon deuxième texte'
    Remarque : "4" et 4 sont deux variables différentes et ne représente pas la même chose.
    On peut également effectuer des « sur ces chaînes » "texte"*2 donne "textetexte" ;
    L'opérateur + permet de concaténer deux chaines : "mon"+"texte" donne "montexte".
    Remarque : Une chaîne peut également être mise entre triples guillemets. Je l'utiliserai pour les commentaires de fonctions.
    Le premier élément de la chaîne ch est ch[0], le dernier peut être obtenu par ch[-1].
    La fonction len permet d'obtenir la longueur de la chaîne : len(ch).
    On peut également extraire une partie de la chaine : ch[3:] qui donne " premier texte", ch[:6] qui donne "Mon pr", voire ch[3:6] qui donne " pr".
  • un booléen : c'est-à-dire que la variable ne peut prendre que les valeurs True ou False (attention à la parenthèse indispensable).
    Ces booléens sont fondamentaux dans les tests et peuvent être combinés par différents opérateurs.
  • un tuple : par exemple a=(1,"mon texte",True) ;
    C'est une suite de valeurs entre parenthèses (même si elles peuvent être omises) qui peuvent être de types différents.
  • une liste (aussi appelé tableau) dans d'autres langages : liste=[10,50,24,15] ;
    C'est une suite de valeurs entre crochets de même nature.
  • un dictionnaire : eleves={"nom":"Doe","prenom":"John"} ;
    Chaque élémént est sous la forme d'une clé à laquelle on donne une valeur.
  • un ensemble : nomEleves={"Antoine","Quentin"} ;
    Les valeurs ne sont pas classées.
  • une fonction : on en verra des exemples plus loin mais certaines peuvent être utilisées dès le début :
    • print(a) : écrit en console la valeur de la variable a ;
    • type(a) : donne le type de la variable a (int, float, string, ...).

Les boucles

Toute répétition d'actions similaires s'effectue sous la forme d'une boucle dans tout langage de programmation.
En python, on distingue deux boucles : for («pour») et while («tant que»).

Boucle for

Ce type de boucles s'écrit sous la forme :

for variable in iterable:
      instructions
Un iterable peut être construit avec la fonction range, une chaîne de caractères, un tuple, une liste, ...
La variable prendra successivement les valeurs de l'itérable et les instructions seront effectuées autant de fois qu'il y a d'éléments dans l'itérable.

Exemple :

montexte="texte initial"
nouveauTxt = montexte[0]
for lettre in montexte[1:]:
      nouveauTxt+="_"+lettre
À la fin de ce programme, nouveauTxt vaut "t_e_x_t_e_ _i_n_i_t_i_a_l".

Boucle while

Une boucle while s'écrit sous la forme :

while condition:
      instructions
où condition est un booléen.
Les instructions sont effectuées tant que la condition vaut True.
Il est donc impératif que la condition puisse évoluer au cours des instructions (l'évolution d'une variable suffit parfois) de sorte qu'elle passe à False. Sinon la boucle devient infinie.

Exemple :

montexte="texte initial"
nouveauTxt2 = montexte[0]
i=1
while montexte[i]!="i":
      nouveauTxt2+="_"+montexte[i]
      i+=1
À la fin de ce programme, nouveauTxt2 vaut "t_e_x_t_e_ ".
On a parcouru chaque lettre de la chaîne de caractère jusqu'à ce qu'on tombe sur un "i".
Si la chaîne n'avait pas eu cette lettre "i", la boucle ne se serait pas arrêtée. On aurait pu le vérifier auparavant ou ajouter une condition d'arrêt supplémentaire (sur la longueur de la chaîne):
while montexte[i]!="i" and i<len(montexte):

Les structures conditionnelles

La structure principale d'une instruction conditionnelle est de la forme

if condition:
      instructions1
else:
      instructions2
Elle peut être déclinée sous la forme
if condition1:
      instructions1
elif condition2:
      instructions2
else:
      instructions3

condition1 et condition2 sont des booléens.

Les conditions doivent nécessairement introduire de nouveaux opérateurs, dits opérateurs de comparaison.

Opérateur < > <= >= == !=
Signification strictement inférieur inférieur ou égal strictement supérieur supérieur ou égal égal différent

Les mots-clés and, or et not
Pour des conditions plus évoluées, on doit utiliser des opérateurs sur les booléens.
Par exemple l'appartenance d'un variable x à l'intervalle [2;3] peut être testée par if x>=2 and x<=8:.
Remarque :En python il existe, pour ce cas, une syntaxe très efficace : if 2<=x<=8:.

On peut aussi utiliser la syntaxe if a is not True.

Les fonctions sont toutes construites suivant le même modèle :

def nom_fonction(arguments):
      instructions
      return variables

Une fonction peut contenir, un, plusieurs ou aucun arguments.
Il est également possible de mettre des valeurs par défaut à un ou plusieurs arguments.
Par exemple

def puissance(x,n=2):
      puis=x**n
      return puis
Dans ce cas puissance(3,4) donnera 81 et puissance(4) donnera 16 (n vaut 2 dans ce cas).

Python contient des fonctions de base :

  • bin, hex pour les conversions en binaire ou hexadécimal;
  • str, int, pour les conversions en chaîne de caractères ou en entier;
  • chr et ord pour le passage d'un nombre au caractère dans la table ascii et inversement;
